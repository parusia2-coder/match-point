# 🏸 동호회 대회용 점수판 설계서 (v2.0)

## 📋 경기 진행 플로우 (6단계)

```
[코트 선택] → [경기 시작] → [점수 입력] → [코트 체인지] → [경기 종료+서명] → [다음 경기 대기]
```

---

## 1단계: 코트 선택 화면 (Side Selection)

### 구현 목표
- 경기 시작 전, 코트 좌/우(또는 상/하) 중 어느 쪽에 어느 팀이 설 것인지 선택
- 드래그 or 버튼 클릭으로 좌/우 팀 배치 결정

### UI 구성 (가로 모드)
```
┌───────────────────────────────────────────┐
│     [코트 X] 경기 준비                       │
│  종목: 남자복식 50대 A급                      │
│  예선/결승: ○ 예선(25점) ● 결승(21점)         │
│                                             │
│  ┌──────────┐    VS    ┌──────────┐        │
│  │ 김철수·이영희 │  ←→  │ 박민수·정하나 │        │
│  │   (좌측)     │      │   (우측)     │        │
│  └──────────┘         └──────────┘        │
│  [↔ 좌우 교체]                               │
│                                             │
│         [🏸 경기 시작]                       │
└───────────────────────────────────────────┘
```

---

## 2단계: 점수 입력 화면 (Scoreboard - 핵심)

### 배드민턴 동호회 규칙
| 구분 | 포인트 | 코트체인지 | 듀스 |
|------|--------|-----------|------|
| 예선 | 25점 단세트 | 13점 | 24:24시 듀스, 26점 선취 or 29:29시 30점 선취 |
| 결승 | 21점 단세트 | 11점 | 20:20시 듀스, 22점 선취 or 29:29시 30점 선취 |

### UI 구성 (가로 전체화면, 태블릿/폰 대응)
```
┌───────────────────────────────────────────┐
│ 코트3 │ 남자복식 50대 A급 │ 예선 25점 │ ↩ │
├───────────────┬───────────────────────────┤
│               │                           │
│   김철수       │        박민수              │
│   이영희       │        정하나              │
│               │                           │
│     12        │         13                │
│               │                           │
│  (좌측/근코트)  │     (우측/원코트)           │
│               │      🔴 서브              │
│     [-1]      │        [-1]               │
├───────────────┴───────────────────────────┤
│ ⏱ 진행시간 04:32  │ 코트체인지: 13점      │
│ [↩되돌리기]                                │
└───────────────────────────────────────────┘
```

### 핵심 로직
1. **터치 영역**: 좌/우 절반 전체가 각 팀의 점수 올리기(+1) 버튼
2. **서브 표시**: 현재 서브 팀에 🔴 표시 (2점마다 서브 교대, 듀스 시 1점마다)
3. **코트체인지 알림**: 
   - 25점 게임 → 13점 도달 시 전체화면 "🔄 코트 체인지!" 표시 + 진동
   - 21점 게임 → 11점 도달 시 전체화면 "🔄 코트 체인지!" 표시 + 진동
4. **듀스 감지 자동화**:
   - 24:24 (25점제) 또는 20:20 (21점제) → "⚡ 듀스!" 표시
   - 2점차 나면 자동 승리 판정 (최대 30점)
5. **자동 경기종료 판정**:
   - 25점(또는 21점) 먼저 도달 시 → 자동 승리 확정 화면 전환
   - 듀스 중 2점차 or 30점 도달 시 → 자동 승리 확정

## 🔧 기술 구현 사항

### 서버 API 추가 필요
1. `PATCH /:tid/matches/:mid/game-type` — 게임 타입(25점/21점) 설정
2. `POST /:tid/matches/:mid/court-change` — 코트 체인지 기록
3. `PUT /:tid/matches/:mid/signature` — 서명 저장 (이미 있음)

### DB 변경 사항
- `matches` 테이블에 `game_type` 컬럼 추가 (default: '25pt' or '21pt')
- `matches` 테이블에 `court_changed` 컬럼 추가 (boolean)

### 프론트엔드 상태 관리
```javascript
gameState = {
  phase: 'setup' | 'playing' | 'court_change' | 'finished' | 'signature' | 'waiting',
  gameType: '25pt' | '21pt',  // 예선/결승
  maxScore: 25 | 21,
  courtChangeAt: 13 | 11,
  courtChanged: false,
  leftTeam: 1 | 2,  // 현재 좌측에 있는 팀 번호
  serving: 1 | 2,   // 현재 서브하는 팀
  startTime: timestamp,
  scores: { team1: 0, team2: 0 }
}
```
