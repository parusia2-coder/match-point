Match Point 단체(협회)
전용 홈페이지 세팅 가이드

누구나 따라할 수 있는 협회/동호회 홈페이지 구축 매뉴얼

개요: 단체 분양 시스템이란?

기존에는 대회를 개설하면 누구나 똑같은 화면을 공유해야 했습니다. 하지만 단체 분양 시스템을 사용하면 우리 협회, 우리 동호회만의 독립적인 전용 홈페이지를 가질 수 있습니다.

💡 도입 시 장점

  • 우리 단체만의 고유한 홈페이지 주소를 가질 수 있습니다. (예: minton-tennis.pages.dev/org/anyang)
  • 우리 단체를 상징하는 로고와 대표 테마 색상으로 홈페이지가 자동 디자인됩니다.
  • 다른 대회와 섞이지 않고, 안양에서 개최한 대회 목록만 깔끔하게 모아서 보여줍니다.

1 관리자 로그인 준비하기

단체를 개설하려면 우선 시스템의 마스터 관리자 권한이 있어야 합니다.

  1. Match Point 메인 화면(minton-tennis.pages.dev)에 접속합니다.
  2. 화면 상단 우측의 [관리자 접속] 버튼을 클릭합니다.
  3. 사전에 발급받은 아이디와 비밀번호를 입력하여 관리자로 로그인합니다.
  4. 로그인 성공 시, 상단에 👑 마스터 계정 이라는 표시가 나타납니다.

2 내 단체(협회) 개설하기

로그인 후, 스크롤을 조금 내리면 홈 화면 중간에 [🏛️ 내 단체 (협회/리그)] 영역이 보입니다. 여기서 설정을 시작합니다.

  1. 우측 상단의 [+ 새 단체 개설] 또는 [🌟 프리미엄 단체 개설 알아보기] 보라색 버튼을 클릭합니다.
  2. 화면에 [프리미엄 단체 개설] 입력 창이 나타나면 아래 정보를 채워 넣습니다.
    • 🏅 단체 종목: 배드민턴 🏸 을 선택합니다.
    • 단체 공식 명칭: 홈페이지 메인에 표시될 이름입니다. (예: 안양시 배드민턴 협회)
    • 전용 영문 주소 (Slug): 우리 안양시 협회 홈페이지로 바로 들어올 수 있는 주소입니다. 영문 소문자와 숫자만 사용해 짧게 적어주세요. (예: anyang)
    • 브랜드 컬러 설정: 홈페이지 메인 테마 색상을 클릭해서 안양시 협회의 상징색(예: 남색, 주황색 등)으로 지정합니다.
  3. 작성을 마친 뒤 창 하단의 [설정 완료] 버튼을 누르면 즉시 새로운 단체 홈페이지가 생성됩니다.

⚠️ 주의사항: 전용 영문 주소 설정

전용 주소는 한 번 정하면 변경이 어려우며, 다른 단체와 중복될 수 없습니다. 소속을 잘 나타내는 간결한 단어(anyang)로 정하는 것을 추천합니다.

3 내 단체 홈페이지 확인하기

개설이 완료되면 홈 화면의 [🏛️ 내 단체] 목록에 안양시 배드민턴 협회가 나타납니다.

  1. 목록에서 우리 단체 카드 안의 [👉 전용 홈 보기] 버튼을 클릭합니다.
  2. 새로운 창이 열리며, 설정한 대표 색상과 타이틀이 적용된 안양시 배드민턴 협회만의 전용 홈페이지가 나타납s니다.
  3. 앞으로 동호회원들이나 안양 지역 참가자들에게 기존의 복잡한 주소 대신, 화면 주소창에 적힌 전용 주소(minton-tennis.pages.dev/org/anyang)만 복사해서 카카오톡으로 공유하시면 됩니다.

4 협회 전용 홈페이지에 대회 올리기

새로 만든 전용 홈페이지에 대회를 노출시키려면, 대회를 개설할 때 우리 단체 이름표를 달아주기만 하면 됩니다.

  1. Match Point 메인 홈(관리자 로그인 상태)에서 [새 대회 개설] 버튼을 클릭합니다.
  2. 기존과 동일하게 대회의 이름과 경기 방식을 입력합니다.
  3. 입력 폼 위쪽의 [소속 협회 선택] 옵션에서 조금 전 개설한 우리 단체 이름을 선택합니다.
  4. 대회를 최종 개설합니다.
  5. 다시 우리 전용 홈페이지 주소로 접속해 보면, 방금 만든 대회가 자동으로 등록되어 있는 것을 확인할 수 있습니다.

💡 응용: 상위 기관 운영 시

소속 협회를 선택하지 않고 대회를 만들면 "일반 공개 대회"로 설정되어 메인 플랫폼에만 노출됩니다. 반드시 소속 단체를 선택해야 전용 페이지에 대회가 등록됩니다.

5 요약 및 향후 업데이트 안내

이제 아래의 흐름대로 운영이 가능합니다.

  • ✅ 관리자: 메인에서 로그인 ▶ 대회 생성 시 우리 협회 선택 ▶ 대회 진행
  • ✅ 참가자: 협회장님이 전달해준 전용 주소로만 접속 ▶ 협회 대회만 보고 신청

🚀 곧 추가될 기능 (업데이트 예정)
향후 [⚙️ 설정 관리] 패널이 열리면, 메인 홈페이지의 메뉴를 내 마음대로 바꾸는 블록형 에디터와 연회비를 관리하는 협회 전속 회원 관리 기능이 오픈될 예정입니다!

디지털로 앞서가는 스마트 체육회, Match Point가 함께합니다.
